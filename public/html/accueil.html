Accueil

Dans le mode automatique vous pouvez demander la localisation d’un bureau dans ce batiment. Une fois la demande faite le drone, qui se trouve dans l’accueil vous guidera jusqu’à l’endroit demandé.

Dans le mode bureau à bureau le drone peut aller d’une pièce à une autre.

Le mode manuel peut vous être utile en cas de défaillance du mode automatique ou pour expérimenter le drone.
