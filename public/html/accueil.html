Accueil

Dans le mode bureau vous pourrez demander la localisation du bureau d'un chercheur dans ce batiment. Une fois la demande faite le drone, qui se trouve à l’accueil vous guidera jusqu’à l’endroit demandé.

Dans le mode bureau à bureau le drone peut aller d’une pièce à une autre.

Le mode manuel vous sera utile en cas de défaillance du mode automatique ou pour expérimenter le drone.
